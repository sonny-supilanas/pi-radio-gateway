Radio Channel MPU 2
1. Setup the Pi on the same network.
2. Install twinkle on Pi using "apt" or use your favorite installation program.

Twinkle is a softphone (https://mfnboer.home.xs4all.nl/twinkle/), configure extension number 16912 on this Pi and make test calls.

3. Copy "data.db" sqlite database file into Pi, note file path.
4. Install Nodered (https://nodered.org/docs/getting-started/local)

Import flow found in this Nodered folder.
